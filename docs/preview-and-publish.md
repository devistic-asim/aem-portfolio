# Preview and publish

| Action | URL suffix | Audience |
|--------|------------|----------|
| Preview | `.aem.page` | Authors / reviewers |
| Publish | `.aem.live` | Public visitors |

## Code vs content

- **Code:** push to GitHub → Code Sync updates both environments’ code
- **Content:** Sidekick Preview / Publish from da.live

## Checklist before publish

1. Preview looks correct on mobile and desktop
2. Nav anchors work (#about, #skills, #projects, #contact)
3. Project and social links open correctly
4. No console errors on the preview URL
5. Lint passes on the branch you merged
