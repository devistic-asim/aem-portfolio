# AEM Sidekick

Sidekick is a Chrome extension for previewing and publishing content.

## Install

1. Install from the Chrome Web Store (search “AEM Sidekick”)
2. Pin the extension
3. Open your da.live document or preview URL
4. Use Sidekick to **Preview** and **Publish**

## Typical flow

1. Edit content in da.live
2. Click **Preview** → check `*.aem.page`
3. Click **Publish** → updates `*.aem.live`

## Tips

- Sidekick needs the site registered (Code Sync + content source)
- Use it on preview URLs and authoring documents
- Pin it so it is easy to find
