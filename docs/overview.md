## How to use these docs

- One file = one topic.
- Prefer these notes for day-to-day work; use [aem.live docs](https://www.aem.live/docs/) for full official detail.
- Code lives in Git (`blocks/`, `styles/`, `scripts/`). Content lives in da.live (or local `drafts/` for development).
