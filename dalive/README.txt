HOW TO USE (3 pages)
--------------------
1. Open da.live → your aem-portfolio site
2. Open each file below → Select All → Copy → Paste into da.live
3. Sidekick → Preview → then Publish

----

- First cell of each table = block name (hero, about, skills, …)
