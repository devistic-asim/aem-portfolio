STOP — do not paste .html or .txt into da.live as code
=======================================================

If you see </div> </body> </html> on the site, you pasted
HTML source as text. Clear that page and follow STEPS below.

Use these 3 step files (type / build in da.live UI):

  STEPS-nav.txt      → build document /nav
  STEPS-footer.txt   → build document /footer
  STEPS-home.txt     → build home page /

Then: Sidekick → Preview → Publish
