# Muhammad Asim — AEM Portfolio

Personal developer portfolio built with **Adobe Experience Manager Edge Delivery Services**.

Dark theme with cyan accents, inspired by modern developer portfolio designs, showcasing Full-Stack .NET / React skills.

## Environments

- Preview: https://main--aem-portfolio--devistic-asim.aem.page/
- Live: https://main--aem-portfolio--devistic-asim.aem.live/
- Local: http://localhost:3000/

## Quick start

```sh
npm i
npm run dev
```

Then open http://localhost:3000/drafts/ (local portfolio content).
