# 09 — Contact, header, and footer

## Contact block

Files: `blocks/contact/contact.js`, `blocks/contact/contact.css`

Rows: Label | Value (or link). Emails and phone numbers auto-link.

## Header

`blocks/header/` loads fragment `/nav` (see `drafts/nav.plain.html`).

Nav structure:

1. Brand section
2. Links list (About, Skills, Projects, Contact)
3. Optional tools section

## Footer

`blocks/footer/` loads `/footer`.

Keep the copyright line short.

## Fragments rule

Fragments are separate documents. Edit `/nav` or `/footer` once; every page that references them updates.
