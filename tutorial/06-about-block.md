# 06 — About block

Files: `blocks/about/about.js`, `blocks/about/about.css`

## What it shows

- Circular photo with cyan ring (placeholder SVG if no image)
- Name
- Social icons (email, GitHub, LinkedIn)
- Section label, heading, bio paragraphs

## Authoring tips

1. First image in the block = profile photo
2. Put social links together in one cell
3. Short label row (“About”) then heading, then paragraphs

## Replace the placeholder photo

Add a headshot in da.live (or reference an image in the about table). Keep it square; CSS crops to a circle.
