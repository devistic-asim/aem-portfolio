# 07 — Skills block

Files: `blocks/skills/skills.js`, `blocks/skills/skills.css`

## What it shows

A wrapping grid of skills. Each item has a name and either:

- An authored icon image, or
- A two-letter badge fallback

## Authoring

One skill per table row. Optional second column for an icon.

This project includes AEM / EDS in the skill list to showcase this site.

## Styling notes

Cyan underline under each icon matches the Dribbble-style reference.
